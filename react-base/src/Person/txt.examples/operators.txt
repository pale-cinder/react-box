    ---Operators

            Spread                          

array of elements               
or object properties

Example:
const newArray = [...oldArray, 1,2]
const newObject = { ...oldObject, newProp: 5 }

    Usage:
const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4];

console.log(newNumbers);

    Usage for objects:
const person = {
    name: 'Max'
};
const newPerson = {
    ...person,
    age:28
}
console.log(newPerson);

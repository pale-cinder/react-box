# Destructuring

Extract array elements or object properties and store them in variables

### Array Destructuring

```javascript
[a,b] = ['Hello', 'Max']
console.log(a) //Hello
console.log(b) //Max
```

### Array Destructuring

```javascript
{name} = ['Hello', 'Max']
console.log(a) //Hello
console.log(b) //undefined

### Examples:
const numbers = [1, 2, 3 ]
```
